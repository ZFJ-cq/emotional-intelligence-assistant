## 1. 产品概述

童话算术闯关小游戏是一款面向低年级儿童的浏览器端算术练习游戏，融合经典童话故事主题，通过趣味闯关方式激发孩子的数学学习兴趣。玩家答对算术题即可解锁下一关卡，体验白雪公主、小红帽、灰姑娘、冰雪奇缘、爱丽丝梦游仙境五大童话世界。

- 目标用户：小学低年级学生（6-10岁）
- 核心价值：将枯燥的算术练习转化为有趣的童话冒险体验

## 2. 核心功能

### 2.1 用户角色
无需注册，直接开始游戏

### 2.2 功能模块
1. **开始页面**：游戏标题、开始按钮、童话风格装饰
2. **游戏主页面**：算术题目、答题输入、角色头像、关卡进度、得分、生命值
3. **关卡切换页面**：关卡解锁动画、新关卡介绍
4. **胜利/失败弹窗**：游戏结果展示、重新开始

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

| 开始页面 | 角色展示 | 五大童话角色头像轮播 |
| 游戏主页面 | 算术题目区 | 随机生成100以内加减乘除题目 |

## 3. 核心流程

玩家打开游戏 → 看到童话风格开始页面 → 点击开始 → 进入第1关（白雪公主）→ 系统随机出算术题 → 玩家输入答案 → 判断对错 → 正确：得分增加、播放正确动画 → 错误：生命值减少、播放错误动画 → 每关答对5题解锁下一关 → 5关全部通过 → 游戏胜利弹窗 → 生命值归零 → 游戏失败弹窗

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：温暖金色(#FFD700) + 梦幻紫(#9B59B6) + 草地绿(#2ECC71)
- 辅助色：每个关卡有专属配色（白雪公主-红蓝、小红帽-红白、灰姑娘-蓝银、冰雪奇缘-冰蓝白、爱丽丝-紫粉）
- 按钮风格：圆角3D立体按钮，带阴影和悬停动画
- 字体：使用 Google Fonts 的 ZCOOL KuaiLe（站酷快乐体）作为标题字体，Noto Sans SC 作为正文字体
- 布局：居中卡片式布局，童话故事书翻页感
- 图标/emoji：使用童话主题emoji（👑🍎🐺👠❄️🐇）

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 开始页面 | 英雄区域 | 渐变星空背景、童话城堡剪影、大标题动画、闪烁星星 |
| 游戏主页面 | 题目卡片 | 圆角卡片、角色头像、大号算术题、输入框、提交按钮 |
| 游戏主页面 | 状态栏 | 关卡进度条、爱心生命值、金币得分 |
| 游戏主页面 | 反馈层 | 正确：绿色对勾+星星飞散；错误：红色叉号+抖动 |
| 关卡切换 | 过渡动画 | 故事书翻页效果、新角色登场 |
| 结果弹窗 | 胜利 | 金色边框、星星评级、彩色纸屑 |
| 结果弹窗 | 失败 | 柔和色调、鼓励文字、重试按钮 |

### 4.3 响应式设计
- 桌面优先设计，同时适配平板和手机
- 触屏优化：按钮足够大，输入框易于点击
- 最小宽度支持 320px
